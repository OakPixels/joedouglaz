Static builds (optional database)


These web builds are purely HTML, CSS and PHP.

They feature MySQL databases and offer total flexibility in the look and function of your site.

However without content management like Wordpress they can only be updated by someone who can code.

Updates can be arranged through myself for a fixed fee.

My build for ADI Theory is an example of a static build. It allows students to submit answers and keeps track of how far they are in the course.


Dynamic Builds - Web Apps


Are your excel spreadsheets boring, confusing or messy? I can tidy them up and visually display the data in a great looking web application.

Alternatively I can link a database to your web app and you can get rid of the excel spreadsheet all together!

This has been successfully deployed in my marina software solution below.

Length values were passed into the web app and applied to the length of the boats on the map.

Data was encrypted and logins secured through Heroku.

Updating the app was as simple as sending an email. Just email your excel file and the web app will take care of the rest.
